# AngkorNYC
> App that allows users to order foods online from a restaurant which is built based on Ember.js
>
> [**Demo**](https://arunras.github.io/angkornyc/)
## Setting Up
**Install Ember**
```bash
$ npm install -g ember-cli
```

**Create New App**

```bash
$ ember new AngkorNYC
```

**Start server**

```bash
$ ember server
```
